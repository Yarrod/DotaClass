Sources

Webapp: Flask app source
Modeling: VGG16 model transfer learning
Project repo: My github repo with the project

Classification process

  • download hero images from google using web crawler
  • preprocess images to be consistent with VGG16 model
  • retrain final layer of the network (retraining more fully connected layers proved to be worse with given amount of data)
  • serve the model using flask and gunicorn for parallel requests
